# BCOG-200-Final-Project

## Project Description
This project will be a simple Python program that measures a user's reaction time. Reaction time experiments are commonly used in cognitive science to measure how quickly someone responds to a stimulus. In this program, the user will complete several trials where they must respond as quickly as possible after a signal appears. The program will record the reaction time for each trial and store the results in a list. At the end of the experiment, the program will calculate and display the average reaction time.

## Planned Functions

### run_trial()
This function will run a single reaction time trial. It will wait for a short delay, display a signal for the user to respond, and record the user's reaction time.

### run_experiment()
This function will run multiple reaction time trials using a loop. It will store the reaction times from each trial in a list.
